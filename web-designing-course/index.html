Our 6-Month Web Design Curriculum

We're big believers in learning by doing. Every month, you'll build a real project that goes straight into your portfolio. Just hover (or tap) over any month below to see exactly what we'll cover.

  • What is web design? – Difference between UX, UI, and visual design
  • Color psychology – How colors influence emotions & actions
  • Typography basics – Font pairing, hierarchy, readability, Google Fonts
  • Layout & composition – Grid systems, white space, visual balance
  • Figma introduction – Setup, frames, shapes, text, and components
  • Wireframing & prototyping – From sketch to digital, linking pages, adding interactions
  • Design systems – Consistent colours, fonts, spacings
  • Project: Build a complete low‑fi prototype for a landing page
  • Document structure – DOCTYPE, head, body, metadata
  • Semantic HTML – header, nav, main, section, article, footer
  • Text formatting – headings, paragraphs, lists, emphasis
  • Links & images – absolute/relative paths, alt attributes
  • ARIA roles & attributes – making sites accessible
  • HTML5 form enhancements – validation, datalist, progress
  • Microdata & schema – adding structured data to HTML
  • SEO best practices – title tags, meta descriptions, heading hierarchy
  • Project: Build a fully accessible and SEO‑friendly product page
  • CSS syntax – selectors, properties, values
  • Box model – margin, padding, border, content
  • Colours & backgrounds – gradients, shadows, opacity
  • Typography – font‑family, size, weight, line‑height
  • Flexbox – one‑dimensional layouts
  • CSS Grid – two‑dimensional layouts, grid‑template‑areas
  • Responsive units – rem, em, vw, vh, %, calc()
  • Transitions, transforms & keyframe animations – hover effects, scaling, rotation
  • Project: Build a responsive product grid with animated hover states
  • Mobile‑first strategy – why it matters for SEO & UX
  • Bootstrap 5 grid system – containers, rows, columns
  • Bootstrap components – navbar, carousel, cards, modal, forms
  • Bootstrap JS components – tooltips, popovers, dropdowns
  • Performance optimisation – image compression, lazy loading
  • Core Web Vitals – LCP, FID, CLS explained and how to improve them
  • Project: Create a fully responsive company homepage using Bootstrap and audit it for speed >90
  • User research – personas, user journeys, empathy maps
  • Information architecture – site maps, navigation structures
  • Interaction design – micro‑interactions, feedback, affordance
  • Building a design system – colours, typography, spacing, components
  • Figma advanced – auto layout, constraints, variants
  • Interactive prototypes – advanced triggers, animations
  • Project: Redesign an existing website using a UX‑first approach + create a complete design system
  • Choose a client – real or mock brief (e‑commerce, agency, portfolio, etc.)
  • Plan & research – competitor analysis, user personas, content strategy
  • Wireframe & prototype – create high‑fidelity Figma designs
  • Build HTML & CSS – code the design using all skills learned
  • Finalise project – polish, test across devices, fix bugs
  • Deploy online – Netlify, Vercel, or GitHub Pages
  • Portfolio presentation – present your work to the class (mock client pitch)
  • Career session – CV & portfolio review, interview tips, job search strategies
  • Certification ceremony – receive your Martian Institute certificate
